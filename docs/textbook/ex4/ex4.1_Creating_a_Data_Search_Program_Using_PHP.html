演習４の手順：

例題のPHP（ダウンロード用URL）は、demoデータベースから3つのテーブルを繋ぎ合わせPDBID, Resolution, Protein name, Organismの列を取り出して表示させるものである。

 

(0) demo.sqlをmysqlにインポートしていない場合は、ファイルをダウンロードしてインポートする。

mysql -uuser -ppassword < demo.sql 

 

(1) 例題のPHP（ダウンロード用URL）を./BioDB/html/のディレクトリに作成してください。

 

(2)  WEB ブラウザ（Firefox等）を起動し、 URL 入力欄に以下の URL を入力しEnter キーを押し、実行結果を確認する。

URL：http://localhost/search_pdb.php

(http://localhost/examples/search_pdb.phpに元々準備してあります)

※phpはWEBサーバーを介していないと動作しません。必ずWEBブラウザを通してURLからアクセスしてください。
 

※よく出るエラーに関して

【デバッグ】よく見るエラーと対処法を参考に修正してみてください。もちろん質問も歓迎します。

 

(3) 例題の出力を確認したら、自作したデータベースを検索するPHPプログラムを作成し、動作確認をしてください。

各自が求める検索ができるようにSQL文を工夫してみて下さい。

また、データベース名とSQL文を入れるとテーブルを表示できるPHPを準備しております。ダウンロードして「./html/」以下に置くと「http://localhost/check_query.php」でアクセスできます。活用してみて下さい。（演習１の時に出しといてよとは言わないで下さい。）

 

 

(4)【課題】自作したデータベースを検索するPHPプログラムを「演習4課題提出先」にアップロードせよ。
