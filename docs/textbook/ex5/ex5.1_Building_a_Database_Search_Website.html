演習5ではHTMLとPHPを組み合わせたデータベース検索WEBサイトを構築する。

 

以下の流れで、まず例題（ダウンロード用URL）を作成してから、自作のWebサイトの構築を行ってください。

 

  1. 例題のGETメソッドで受けた引数を検索条件として、データベース検索をして、その検索結果を動的に表示させるPHPプログラムを作成する。
  2. 例題のフォームに入力したキーワードを引数として先に作成したPHPに送る検索用のHTMLページを作成し、 1.のPHPと合わせてデータベース検索Webサイトを構築する。
  3. 自分で作成したデータベースを使って、データベースを検索し表示させるWebサイトを構築する。
 
演習5の手順：

(1) 例題のlogin.phpとsearch_get.phpを./BioDB/html/ディレクトリ以下に作成する。

※login.phpはmysqlへのログイン用のphpです。search_get.phpの中でrequire(login.php)と呼び出されています。DB名は各自変更が必要ですので確認しておくこと。

 

(2) WEB ブラウザを起動しURL 入力欄に以下の URL を入力しEnter キーを押し、実行結果を確認する。

URL: http://localhost/search_get.php

 

(3) GETメソッドの引数は次のように~~~.php後に「?」をつけて受け渡ことができる。
引数を下記の例のようにいろいろ入力してみて、検索が行われて結果の表が動的に変化するのを確認してください。

URL:  http://localhost/search_get.php?id=1AGW
URL:  http://localhost/search_get.php?res=2
URL:  http://localhost/search_get.php?org=Human&res=2.5

 

(4) 例題のsearch_form.htmlを./BioDB/html/ディレクトリ以下に作成する。

　

(5) WEB ブラウザを起動しURL 入力欄に以下の URL を入力しEnter キーを押し実行結果を確認する。ダブルクリックで開かないように注意。WEBブラウザは立ち上がりますが「search」を押してもPHPが動きません。WEBサーバーを介していないため。

URL: http://localhost/search_form.html

※Webフォームに何も入力せずに「search」ボタンを押すと(2)と同様の結果が表示される。検索ワードを入れて「search」ボタンを押すと、検索結果の表が検索ワードによってうまく絞り込まれていることを確認してください。

 　

(6) 例題の出力を確認したら、自作したデータベースを検索するPHPプログラムを作成してください。

    • まずは、例題を自作データベースに変更するぐらいから始めてみてください。
    • 過去の優秀作品などを参考に改良をしてみて下さい。ただし、優秀作品だけあってレベルは高いです。
