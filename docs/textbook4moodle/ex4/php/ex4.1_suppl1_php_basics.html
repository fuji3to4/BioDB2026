PHPは、スクリプト言語Perlや、さらにはC言語の流れを組むスクリプト言語です。

１．変数

変数とは、名前をつけたデータの格納庫です。変数に名前を定義するときは、名前の前に$記号をつけて変数名を指定します。
PHPの変数宣言は次のように行います。

$変数名 = 値;

たとえば、中身が数値の変数 $a を作成する場合は、次のようになります。

$a = 2;

また、中身が文字列である変数 $b を作成する場合は、値となる文字列を引用符（"）で囲みます。

$b = "こんにちは。";

変数名としてつけることができる文字と記号は、英字、数字、_（アンダースコア）です。
次の例は、変数名として扱えます。

$_abc
$Abc
$abc_3

一方、次の例は変数として利用できません。

$_?     // アンダースコア以外の記号を使用する
$3rd    // 数字が1文字目にくる変数名

変数に値を代入するとき、計算結果を代入することができます。

$price = 20000 + 30000;

この場合、$price の値は 50000 となります。

文字列を連結して代入する場合は、以下のように「.」で連結します。

$hello = "こんにちは。" . "お元気ですか？";

$hello の値は "こんにちは。お元気ですか？" となります。

 

２．定義済み変数

以下の変数は、PHPに準備されている変数で、複数の値を持つ「配列」という形になっています。

主な定義済み変数

  • $_GET  : クライアントからGETメソッドで渡されたデータ
  • $_POST : クライアントからPOSTメソッドで渡されたデータ
  • $_FILES: POSTメソッドで送信（アップロード）されたファイル
