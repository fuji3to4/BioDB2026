PHPの型

PHPが扱うデータを分類すると、次のように大きく3種類に分けられます。

 

◆スカラー型

整数： int, integer
整数で小数部を持たない数値。四則演算など算術計算ができる。10進数の他、8進数、16進数をサポートしている。

$val = 99;    // 10進数の例
$val = 077;   // 8進数の場合は先頭に0をつける（077は10進数の63）
$val = 0xf;   // 16進数の場合は先頭に0xをつける（0xfは10進数の15）

文字列： string
文字の連続で、アルファベット、数字、記号などを含む。

$name = "藤井聡";
$comment = "私は".$name."です。";   // 「.」で文字列を連結することができる
$comment = "私は${name}です。";     // ""の中に{}を用いて変数の値を展開できる

※文字列の中で変数を展開する場合、外側の引用符は二重引用符（"）を使う必要があります。

浮動小数点数： float, double, real
小数部を持つ数値。四則演算など算術計算ができる。

論理型： boolean, bool
TRUEとFALSEの値を持つ型。真偽を表現するときに使用する。

 

◆複合型（スカラー型を組み合わせることによって構成された型）

オブジェクト： object
classとして定義した独自型。その中に任意の複合型の変数を持つことができる。

配列： array
任意の数の複合型を格納するための領域。C言語と同様に[0]から値を格納できるが、キーを文字列で指定する連想配列を作ることも可能。
[書式] $<配列名>[<キー>]

$a[1] = 50;
$b["web"] = "www.bio.kyutech.ac.jp";   // キー名は文字列でも可

配列の初期化：

$a = array(10, 20, 30);   // $a[0]～$a[2]にそれぞれ10,20,30が設定される

 

◆特別型（上記以外で分類できない例外的なデータ）

リソース： resource
ファイルにアクセスするためのファイルポインタなど、コンピュータ上のリソースを扱うための型。

ヌル： NULL
空を意味する型・状態。
